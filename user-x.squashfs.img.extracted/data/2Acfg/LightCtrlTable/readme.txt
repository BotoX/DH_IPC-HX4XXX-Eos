灯光控制说明：
1、针对复杂云台，灯光控制只有1位表示；
2、针对简单云台，均需要切换为采用3位表示。

A、x的定义：
[0]：不支持红外灯，
[1]：亮度可调的红外灯
[2]：亮度不可调的红外灯
[3]：激光
[4]：白光
[5]：光轴可调激光
[6]：复合灯（近1：红外，近2：白光，远：红外）
[x]

B、具体类别：
[x00]：云台枪型号
[x10]：PTZ型号(大云台)
[x20]：29型号x类型灯
[x40]：49型号x类型灯
[x50]：59型号x类型灯
[x60]：6A8型号x类型灯
[x61]：6A9型号x类型灯
[x62]：6C1型号x类型灯
[x63]：6C3型号x类型灯
[x80]: 傅立叶型号x类型灯（8寸球）
[xFF]
