Parameter Function
Path 1、Here you can set Device record and snapshot storage setup.System can record according to the event types (General, motion detect and alarm) in the Schedule interface. You can check the box here to storage the corresponding record or snapshot picture.
2、There is local SD card, FTP or NAS storage setup for you to choose .

FTP Please select the FTP in the storage destination interface. Emergency save in the local can save the record file or snapshot picture in the local SD card when the network is offline or malfunction.
NAS Please select the NAS in the storage destination interface. When selected the NAS storage, you can save files to the NAS server.

Note： FTP and NAS both belong to network storage, they can not work at the same time.
