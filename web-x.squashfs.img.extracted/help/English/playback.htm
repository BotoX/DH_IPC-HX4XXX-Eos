Playback back list

Comment： Only the equipments which support these scales have the function.

Playback function buttons

  • Play - This button means pause play or there is no playing record. Click it again to switch to normal play status.

  • Stop - Click it to stop playback.

  • Play frame by frame – Click it to go to the next frame to play.

  • Slow play – Click it to lower the play speed.

  • Fast play - Click it to speed up the play speed.

  • Mute - The system is mute when you see current button. Click it again, the audio restores normal status.

    •  

    Note: Please pause current playback operation if you want to use frame by frame play function.

File Type

You can choose record or snap to playback.

File list options

In the calendar, the blue highlighted color means there is record file in current date and is current selected. Double click a file name in the list, system begins playback. You can view file size, start time and end time. You can also see there are several options for you such as download, download more, watermark, back and search.

Search：It is to search the record files in current day from the start time to the end time you input here.

Download：When record type is dav,click download button, you can download specified file(s) to the local end; When record type is MP4,click download button, you can download specified file(s) to Camera -> Video -> Path.

Back：Click it, you can go back to the Calendar interface to select time again to operate.

Note: Record does not support download and playback operation at the same time.

Record Type/Snap Type

After you checked the file type, you can just show the corresponding files in the process bar or the file list.

Playback Cut

When you selected start time and end time, click the putton to save file to Camera -> Video -> Path.

Note: Please pause current playback operation if you want to use playback cut function.

Process bar time format

  •   24-h button - Click it, system displays the process bar in 24-h mode.

  •   2-h button - Click it, system just displays a 2-hour period bar (0-2 o’clock, 2-4 o’clock, 4-6 o’clock and etc) bar for you reference. For example, the process bar ranges from 8 to 10 o’clock if you record file time is recorded at 9 o’clock.

  •   1-h button - Click it, system just displays a 1-hour bar for you reference.

  •   30-m button – Click it, system just displays a 30-minute bar for you reference.
