Alarm introduce

The alarm module is for you to subscribe the alarm event. When an alarm you subscribe occurred, you can view the alarm information on the right pane. You can check the box to enable prompt or alarm sound so that there are alarm prompt information or sound to alert you.

Parameter

 

Alarm type： Please check the box here to select the corresponding alarm event.

pour： Once the even you subscribed occurred, there is no video when you are in alarm interface. You can view the corresponding alarm information in the alarm list on the right pane.

Audio： You can check the box here and then select the audio file name. When an alarm occurs, system auto generates alarm audio to prompt you.
