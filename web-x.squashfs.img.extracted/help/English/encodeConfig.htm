Video Bit Stream

Code-Stream Type ■ System supports active control frame function (ACF). It allows you to record in different frame rates. For example, you can use high frame rate to record important events, record scheduled event in lower frame rate and it allows you to set different frame rates for motion detection record and alarm record.
■ It includes general stream, motion detect stream and alarm stream. You can select different encode frame rates form different recorded events.
Encode mode ■ H.264 ： Main Profile encode mode.
■ H.264H ： High Profile encode mode.
■ H.264B ： Baseline Profile encode mode.
■ MJPEG ： In this encode mode, the video needs larger bit stream to guarantee the video definition. You can use the max bit stream value in the recommend bit to get the better video output effect.
Resolution There are multiple resolutions. You can select from the dropdown list.For each resolution, the recommended bit stream value is different.
Frame Rate（FPS） 50Hz: 1～25f/s，60Hz: 1～30f/s.
Bit Rate Type ■ There are two options: VBR and CBR. Please note you can set video quality in VBR mode.
■ In MJPEG encode mode, the bit stream control mode shall be CBR only.
Bit Rate In the VBR mode, it is the threshold value of the bit stream. In the CBR mode, it is a fixed value.
I Frame Interval Here you can set the P frame amount between two I frames. The value is 150. Recommended value is frame rate *2.
SVC frame rate can be encoded by layer. It is a flexible encoding method. By default, it is 1 as 1 layer. You also can set 2/3/4 layers.
Watermark ■ This function allows you to verify the video is tampered or not.
■ Here you can select watermark bit stream, watermark mode and watermark character. The max length is 128-digit. The character can only include number, character, underline and hyphen.

Snapshot

Snapshot Type ■ General snapshot is to snap during the specified period.
■ Activation snapshot is to snap after motion detect, camera masking or local alarm occurred.

Activation snapshot can become active once you setup meet the following items:
1.The action is in the specified period.
2.The motion detect, camera masking or local alarm and their snapshot functions are enabled.

Image Size It is the same with the resolution of the main stream.
Image Quality The image quality can be best, better, good, bad, wrose or wrost.
Interval It is to set snapshot frequency.

Overlay

Privacy Masking ■ Here you can privacy mask the specified video in the monitor video.
■ System max supports 4 privacy mask zones.
Channel Title ■ You can enable this function so that system overlays channel information in video window.
■ You can use the mouse to drag the channel title position.
Time Title ■ You can enable this function so that system overlays time information in video window.
■ You can use the mouse to drag the time title position.
Text Overlay ■ You can enable this function so that system overlays text information in video window.
■ You can use the mouse to drag the text overlay position.Text Align consists of left and right.
Picture Overlay ■ You can enable this function so that system overlays picture information in video window.
■ Click Upload picture to overlay local picture to video window. Drag the yellow frame to adjust overlay position.
Refresh After you successfully set and saved the privacy mask zone, channel title, time title and location, you can click the refresh button to see the effect.

Note: The text overlay and picture overlay can not be valid at the same time.
