IVS—Normal back list

Global Setup


IVS Setup

Set module Click it to make inteligent algorithm understand the background again. Typically after the camera strenuous or move exercisely, you need it to improve the detection rate of inteligent function.
Apply Scene Select Normal
Jitter Rate Here you can set device de-jitter rate,The default setup is 50.
Calibration area Please draw a horizontal ruler and a vertical ruler
Actual length Please enter the actual situation
Calibration Calculated according to the system,compared with the actual length of the reference

Rule Setup

Add rule Click it to add a rule to the selected scene.
Rule enable Check the box here to enable the rule.
Rule name Move you mouse to the rule name column to set a name
Rule type Move you mouse to the Rule type column to set the rule type. The option includes: Tripwire/Intrustion/Cross warning zone/Abandoned Object/Missing Object/Loitering Detection/Parking Detection/Directional Flow Violation Detection/Fast-Moving/Crowd Gathering Estimation.
Draw rule Click button,to draw current rule
Delete rule Click button,to delete current rule

Area and Property Setup

Object type supported You can select an object type here. The default setup is unknown.
Detect action list ■In area: When the tracking object is in the detect zone and the stay time is more than the time threshold you set here, system can activate an alarm.
■Through area: System can generate an alarm when the tracking object is crossing the limit you set here.
Direction It is for the Through area option in the Detect action list item. It is to set the cross direction. The option includes: Enters,Exits,Enter&Exit.
Max target No It is for the In area option in the Detect action list item. It is to set the alarm activation amount in the zone. System will not generate an alarm when the object amount in the zone is more than the threshold you set here.
Min target No It is for the In area option in the Detect action list item. It is to set the alarm activation amount in the zone. System will not generate an alarm when the object amount in the zone is less than the threshold you set here.
Min last time It is for the In area option in the Detect action list item. Here is to set the min time between the target in the zone and an alarm activation occurrence.
Report interval It is for the In area option in the Detect action list item. It is to set alarm period. System just alarms once if the value is 0 here.
Target filter Check the box here to enable this function and then click button 'draw', you can set the target filter model in this scene for this rule. Click the button 'clear', you can remove selected target filter model

Alarm Activation Setup

Rule Working period Here you can set alarm arm and disarm period. Please click the Setup button to set.
Record Check the box here so that the device can record when alarm activation occurred.
Relay-out Check the box here so that the device can output an alarm when alarm activation occurred.
Send e-mail ■Check the box here so that the device can send out an email when alarm activation occurred
■Please note current function is null when the device is offline, or IP conflict occurred.
Snapshot Check the box here so that the device can snapshot when alarm activation occurred.

Parameter

Scene Please select a scene from the dropdown list you want to set.
Detect balance There are three modes: compromise mode, omission detect less and error detect less
Sensitivity The level ranges from 1 to 10. The default setup is 5. The higher the sensitivity is, the easier for the device to detect the low contrast object and small object. At the same time, the error detect may become more.
Anti-Disturb Enable It is to control the random disturbance in the scene.
Target filter Click button 'draw'，you can set the target filter model in this scene for all rule. Click the button 'clear'，you can remove selected target filter model
Draw Detect Region Click button 'Add Detect Region', you can draw a detection zone.
Draw Exclude Region Click button"Add Exclude Region"，you can draw the privacy mask (shield) zone.

Comment: Only the equipments which support these scales have the function.

Global Setup

Apply scene Select Face Detection.
Main Direction ■draw ：Click button,and draw flow direction line in the tip region.
■clear ：Click button,delete the current flow direction line.
Pitch angle ■positive value ：When the pitch angle is a positive value K,it means the pitch up angle is K degree,such as the illustrations of the page.
■negative value ：When the pitch angle is a negative value K,it means the pitch down angle is K degree,such as the illustrations of the page.
left and right angle ■positive value ：When the left and right angle is a positive value K,it means the left angle is K degree,such as the illustrations of the page.

Parameter

Face Detection Zone ■Add detect region ：Click button 'Add Detect Region', you can draw a detection zone.
Detect width range ■Maximum size ：Show the maximum size of the face which can be detected,it will not be detected if the size is more than the big one.
■Minimum size ：Show the minimum size of the face which can be detected,it will not be detected if the size is less than the big one.
■Draw ：Click button, you can draw a zone in the tip region.
■Delete ：Click button，you can delete the drawed zone.
Accuracy The level ranges from 1 to 10. The default setup is 8. The higher the accuracy is,the error detect may become less.
Sensitivity The level ranges from 1 to 10. The default setup is 8. The higher the sensitivity is, the easier for the device to detect the low contrast object and small object. At the same time, the error detect may become more.
