UPnP Introduce

 

You can use the UPnP protocol to establish a forwarding relationship. The camera port is the internal port and the router port is the external port. You can access the external port to visit the IP camera.

Note

 

Please check the box here to enable the UPnP function. The IPC now supports ，UPnP protocol. Now the My Network Places can detect the IPC if you are using the Windows XP or Vista OS.

In the Windows OS, From Start->Control Panel->Add or remove programs. Click the “Add/Remove Windows Components” and then select the “Network Services” from the Windows Components Wizard. Click the Details button and then check the “Internet Gateway Device Discovery and Control client” and “UPnP User Interface”. Please click OK to begin installation.
