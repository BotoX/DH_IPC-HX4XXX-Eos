Parameter Function

In the schedule interface, you can add/remove three types of record plan or snapshot plan. There are six periods in one day. System can not enable the record function in the specified period if the record function is disabled in the record control interface. You can view the current time period setup from the color bar.

there into：

■ Green color_stands for the general record/snapshot.

■ Yellow color_stands for the motion detect record/snapshot.

■ Red color_stands for the alarm record/snapshot.

Holiday setup interface is for you to set holidays. If snap/record on the holiday interface is enabled, for those dates set in the holiday interface, system enables snap/record according to the holiday setup.
