Parameter Function
Audio Channel Audio channel shows all audio channels of current device. Select different audio channels and current channel info will be displayed below.
Sensitivity Level 1-100 adjustable, the lower the value, the higher of the difference in input volume change will be than continuous environmental volume. This can be judged as abnormal audio. User shall adjust it according to actual environment.
Anomaly Threshold Level 1-100 adjustable, used to set filtered environmental sound intensity. The higher the noise, the higher the value shall be set. User shall adjust according to actual environment.
