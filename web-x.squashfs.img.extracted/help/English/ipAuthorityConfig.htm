Function

 

Trusted Sites：The IP listed below can login the device. Once you check the white list function, only the IP listed in the trusted sites list can login the specified device. All the users can login the device if you do not check the trusted sites box here.

The device IP can not listed in the trusted list.

The MAC verification can only be valid when the device IP and the PC IP are in the same IP segment.

Note： When there is WAN access, it shall be limited by the MAC of the router.
