Parameter Function

 

Please check the box to enable WIFI function and then click the Search SSID button. Now you can view all the wireless network hotspot information in a list. Double click an item, you can see the signal intensity, authentication mode. Please note sometimes you need to input the password.

Please check the box to enable WIFI function and then click the Add SSID button to add the SSID you want.Then click the Search SSID button,if you can view the wireless network hotspot you add in the list,you can use this wireless network hotspot as you want.

When you input the password, please confirm that the index of the password shall be the same as that of the router. You can click the Connect directly if there is no need to input the password.

You can view the connected hotspot name, status, IP information in the WIFI working information column. You need to click the refresh button to get the latest information if you want to connect again. The WIFI hotspot connection may take some time, it has relationship with the network signal intensity.
