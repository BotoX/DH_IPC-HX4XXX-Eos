PTZ Settings back list

Comment： Only the equipments which support these scales have the function.

Here is to set PTZ protocol, address, baud rate, data bit, stop bit and parity,Please refer to the following sheet for detailed information.

Parameter Function
Protocol Select the corresponding PTZ protocol.
Address Set corresponding device address.

Note： Please note your setup here shall comply with your device address; otherwise you can not control the device.

Baudrate Select the device baud rate.
Data Bit Default setup is 8.
Stop Bit Default setup is 1.
Parity Default setup is none.
