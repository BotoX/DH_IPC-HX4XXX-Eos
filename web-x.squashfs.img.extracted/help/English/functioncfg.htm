PTZ back list

Preset

Preset No. Set preset No. It becomes valid after you set.
Preset title Set preset name. You can input self-defined preset name here.
Delete Delete corresponding preset.
Save Save current preset setup.
Add Click it to add a new preset.
Clear Clear all presets setup.

Tour

Tour No. System auto allocates.
Tour title Set tour name. You can input self-defined tour name here.
Start Begin tour.

Attention:Please note this function is null if you have not set a tour.

Add Click it to add a new tour.
Preset ■ Click Add button, you can add a new tour. The default preset value is 1.
■ Double click tour No, you can see a dropdown list for you to select.

Attention:Please set presets before you use tour function.

Duration It is to set the staying time of the tour.
Save Click it to save current setup.
Refresh Refresh current interface.

Scan

Scan No. System auto allocates.
Speed ■ Set scan speed here.

Attention:You can not set speed when camera is running scan.

Start ■ Begin to scan.

Attention:Please note this function is null if you have not set a scan.

Set Click set button, you can set left limit and right limit.
Left limit Set camera left limit.
Right limit Set camera right limit.

Pattern

Pattern No. System auto allocates.
Start ■ Begin to pattern.

Attention:Please note this function is null if you have not set a pattern.

Set Click it you can see start/stop record buttons.
Start record Click it to begin recording pattern setup.
Stop record Click it to stop recording pattern setup.

Privacy mask

SN Please select privacy mask number from the dropdown list.
Enable Check the box here to enable privacy mask function.
Draw ■ Click Draw button, you can see edit column. You can drag and release mouse to set privacy mask region and size.
■ You can draw the edit column or you can right click to remove.

Attention:Before the operation, please make sure you have enabled privacy mask function.

Clear Click it to clear all privacy mask setup.

PTZ speed

PTZ speed Here you can set camera PTZ speed.

Idle Motion

Enable Check the box here to enable idle motion function.
Idle Motion ■ Here you can set preset, scan, pattern, tour.

Attention:Please note you need to set the corresponding function first if you want to use it.

Idle time Set idle time.

Power up

Enable Check the box here to enable power up function.
Power up ■ Here you can set preset, scan, pattern, tour and auto.

Attention:Please note you need to set the corresponding function first if you want to use it.

PTZ restart

PTZ restart It is to reboot PTZ.

Default

Default Restore PTZ factory default setup.
