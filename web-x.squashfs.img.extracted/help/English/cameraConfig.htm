Conditions back list

Comment: Only the equipments which support these scales have the function.

Profile

You can select normal, daytime or night mode. After you selected mode, you can set and view the setup and effect of the corresponding mode.

Profile Management

It works with the configuration files. You can select normal/daytime/night mode according to your actual requirements.

Brightness

It is to adjust monitor window brightness via the nonlinear mode. The larger the number is, the bright the video is. Please note the video may become hazy if the value is too high. The recommended value ranges from 40 to 60. The default value is 50.

Contrastness

It is to adjust monitor window contrast. The larger the number is, the higher the contrast is. Please note the video may become hazy if the value is too low. If this value is too high, the dark section may lack brightness while the bright section may over exposure .The recommended value ranges from 40 to 60. The default value is 50.

Saturation

It is to adjust monitor window saturation. The larger the number is, the strong the color is. This value has no effect on the general brightness of the whole video. The recommended value ranges from 40 to 60. The default value is 50.

Sharpness

It is to adjust the sharpness of the video edge. The higher the value is, the clearer the edge is and vice versa. There may be noise if the value is too high. The default value is 50 and the recommended value ranges from 40 to 60.

Gamma

The threshold here is to adjust video brightness via the nonlinear mode to enhance video dynamic display range. The larger the number is, the bright the video is. The default value is 50 and the recommended value ranges from 40 to 60.

Mirror

It is to switch video left and right direction.This function is disabled by default.

Flip

The flip function can turn the monitor video direction. It is off by default.

Anti-flicker

50HZ When the current is 50Hz, it can guarantee there is no cross band when system auto exposures according to the brightness.
60HZ When the current is 60Hz, it can guarantee there is no cross band when system auto exposures according to the brightness.
Outdoor In this mode, you can switch exposure mode to achieve the effect in the corresponding exposure mode.

Exposure

The exposure mode includes: auto, gain priority, shutter priority, aperture priority and manual.

Auto mode The overall brightness of the image can be auto adjusted according to different scene brightness within the range of normal exposure.
Gain Priority The overall brightness of the image can be auto adjusted according to the gain range within the range of normal exposure. If the image brightness is still improper and the gain has reached the threshold, then it will auto adjust the shutter value to make the image normal.
Shutter Priority The overall brightness of the image can be auto adjusted according to the shutter range within the range of normal exposure. If the image brightness is still improper and the gain has reached the threshold, then it will auto adjust the gain value to make the image normal.
Iris Priority An iris value can be set on the web, and the best brightness can be auto realized according to the mode of driving exposure time preferentially and followed by driving gain.
Manual It can manually adjust gain value and shutter value under the mode of manual exposure. Gain range is 0-100 adjustable, shutter value supports 1/100000 and customized range, support long exposure.

Note: System only switches exposure mode when then anti-flicker mode is outdoor.

Auto Iris

You can check the box before ON to enable this function. This function is on by default.

Note: Before the setup, please make sure you have installed the auto iris.

3D NR

3D NR reduction is mainly in view of the frame(frame) for at least two image processing,information between frames between before and after the use of video frames for noise reduction.The larger the number is, the better the results.

BLC Mode

It is the backlight compensation button. This function is for backlight environments.

BLC ■ The device auto exposure according to the environments situation so that the darkest area of the video is clears.
■ Customized BLC：The selected area of the device exposures properly so that it can get the proper brightness.
WDR For the WDR scene, this function can lower the high bright section and enhance the brightness of the low bright section. So that you can view these two sections clearly at the same time.
HLC After you enabled HLC function, the device can lower the brightness of the brightest section according to the HLC control level. It can reduce the area of the halo and lower the brightness of the whole video.

White Balance

It is to set the white balance mode. It has effect on the general hue of the video. This function is on by default.

Auto Auto set white balance, which can automatically realize white balance compensation for different color temperatures to make the image color normal.
Natural It can auto realize white balance compensation to the scene without artificial light, which is to make the image color normal.
Street Lamp It can auto realize white balance compensation to the outdoor scene at night, which is to make the image color normal.
Outdoor It can auto realize white balance compensation to most of the outdoor scenes with natural light and artificial light, which is to make the image color normal.
Manual It can set red and blue gain manually; the value range is from 0 to 100.

Day & Night

It is to set device color and the B/W mode switch. The default setup is on.

Color Camera outputs the color video only.
Auto Device auto selects to output the color or the B/W video according to the device general bright of the video.
B/W The device outputs the black and white video only.

D&N Sensitivity,D&N Delay

Set sensitivity and delay value during day/night switch process. The delay value is 6.

Flash

Manual ■ Single channel IR control device, in the manual mode, IR light is controlled by the value of middle light, range is 0-100, default is 50.
■ Double channel IR control device, in the manual mode, IR light is controlled by near light and far light, range is 0-100, default is 50.
Auto Auto adjust the IR light.
Off Turn off the IR light.

EIS

It can realize electronic anti-jitter function via algorithm of comparing image difference, which has efficiently solved the problem of image dithering during use and makes the picture much clearer.

Defog Mode

The image quality will be lower when the device is exposed to the environment with fog and haze, the image can operate auto de-warp in auto mode. It can also select different intensity manually according to the fog density to adjust the image definition.

Focusing Scale

Comment： Only the equipments which support focusing scale have the function.

Zoom

Zoom out If you press the minus key,the focal length of the video decreases.
Zoom in If you press the add key,the focal length of the video increase.
Speed Speed consists of 1，5，20，100.

Focus

NEAR If you press the minus key,the focal plane of the lens moves to the close shot.
FAR If you press the add key,the focal plane of the lens moves to the long shot.
Speed Speed consists of 1，5，20，100.

Restory All

Restory All,the value of focus and zoom change to 0.

Autofocus

Adjust the focal plane of the lens to make the video in the clearest condition.

Refresh

Obtain the newest condition of focusing scale.
