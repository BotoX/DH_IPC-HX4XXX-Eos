Log Introduce

It is to record user operation information and some system information.

Parameter Function

Parameter Function
Start Time Set the start time of the requested log.(The earliest time is Jan 1st 2000)
End Time Set the end time of the requested log.(The oldest time is Dec 31st2037.)
Type Log types include: system operation, configuration operation, data operation, event operation, record operation, user management, log clear.
Search You can select log type from the drop down list and then click search button to view the list.；You can click the stop button to terminate current search operation. You can view searched items amount and period.
Detailed Information Click one log, you can view its detailed information.
Clear You can click this button to delete all displayed log files. Please note system does not support clear by type.
Backup You can click this button to backup log files to current PC.

Please refer to the following contents for detailed definition of different log types：

The system operation log type includes: "StartUp", "Abort", "Exit", "Restart", "ShutDown", "Reboot", "Upgrade"

The setting operation log includes: "SaveConfig", "RecallConfig","Zoom and Focus".

The data operation includes: "SetWorkDirType", "ClearWorkDir", "HotPlug", "FTP Status", "NAS Status", "RecordMode","SD card error","Disk error".

The event operation includes（record motion detect, camera masking and local alarm event）: "EventStart", "EventStop"

Record operation includes: "FileAccess", "FileAccessError", "FileSearch"

Account（Record account modification and user login/logout.）： "LogIn", "LogOut", "AddUser", "DeleteUser", "ModifyUser", "AddGroup", "DeleteGroup", "ModifyGroup"

Clear log: ClearLog
