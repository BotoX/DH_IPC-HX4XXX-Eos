Video Detect back list

The video detect include motion detect and the video tamper

Parameter Function
Enable You need to check the box here to realize the motion detect or video tamper alarm.
Working period 1.The alarm function becomes activated in the specified periods.
2.There are six periods in one day. Please check the box to enable corresponding period.
3.Select date. If you do not select, current setup applies to today only. You can select all week column to apply to the whole week.
4.Click OK button, system goes back to motion detection interface, please click save button to exit.
Anti-dither System only memorizes one event during the anti-dither period. The value ranges from 0s to 100s.
Area Click Set button to go to the corresponding interface. It is for you to set valid motion detect area. You can set sensitivity, area size respectively (The motion detect can be easily triggered if the sensitivity is bigger or the area is smaller.). The default valid area is the whole video area.
Record Check the box so that system auto activates motion detection channel to record once an alarm occurs. Please note you need to go to “ Set->Storage->Schedule ”to set current channel as general record.
Record Delay System can delay the record for specified time after alarm ended. The value ranges from 10s to 300s.
Relay-out Check the box here to enable alarm activation function. You need to select alarm output port so that system can activate corresponding alarm device when an alarm occurs.
Alarm Delay System can delay the alarm output for specified time after an alarm ended. The value ranges from 10s to 300s.
Send Email If you enabled this function, System can send out email to alert you when alarm occurs and ends. Please note you need to go to“ Set->Network->SMTP(Email) ”to set your own email address.
PTZ Please check the box here to enable this function. Here you can set PTZ movement when an alarm occurs. Such as go to preset x when there is an alarm. The event type includes: preset, tour and pattern.
Snapshot You need to check the box here so that system can backup motion detection snapshot file. Please note you need to go to the“ Set->Storage-Schedule ”to set snapshot period.
