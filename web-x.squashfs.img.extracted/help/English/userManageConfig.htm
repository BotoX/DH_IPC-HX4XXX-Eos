Note

 

For the version information, and the other icons excluding the alarm in the live interface are read-only.

Only the user of account rights can implement the user management operation.

■ 1) For the character in the following user name or the user group name, system max supports 15-digits. The valid string includes: character, number, and underline.

■ 2) The password length ranges from 0 to 32. It can only include the number and characters. The user can modify the password of other users.

■ 3) The factory default amount of the user is 18 and the factory default amount of the user group is 8.

■ 4) User management adopts group/user modes. The user name and the group name shall be unique. One user shall be included in only one group.

■ 5) Current login-in user can only change his own rights.

There are one default users admin。admin have administrator right.

The rights of the user can not exceed the rights of the user group it belongs to.
