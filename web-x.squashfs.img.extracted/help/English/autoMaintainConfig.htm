Auto Maintain Introduce

Here you can select auto reboot and auto delete old files interval from the dropdown list. If you want to use the auto delete old files function, you need to set the file period.

Parameter

 

Check “Auto Reboot” box to enable this function and then set the reboot time.

Check “Auto Delete Old Files” box to enable this function and then set the period. The time period ranges from 1 to 31 days.
