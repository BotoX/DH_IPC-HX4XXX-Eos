Dial Setup

Wireless Network Type ■ WCDMA signal——Dial, message, incoming call(Only for China Unicom)
■ TD-SCDMA signal——Dial, message, incoming call(Only for China Mobile)
■ EVDO signal——Dial(Only for China Telecom)
■ TD-LTE signal——Dial
■ FDD-LTE signal——Dial(For China Telecom/China Unicom)
■ CDMA1X signal——Message, incoming call(Only for China Telecom)
■ Auto Mode——Dial, message, incoming call(Recommended)
Enable Check the box here to enable 4G module
APN For the geenral cards, APN of China Telecom is ctlte，China Mobil is cmnet，China Unicom is 3gnet. It may vary due to different cards.
Authorize Mode For the general cards, the authentication mode of China Telecom is CHAP or PAP. There is no authentication mode for China Mobile or China Unicom. It may vary due to different cards.
Dial-up Number For the general cards, the dial number of China Telecom is #777. For China Mobile, it is *98*1#. For China Unicom, it is *99#. It may vary due to different cards.
Username For the general cards, the user name of China Telecom is ctnet@mycdma.cn or card. There is no user name for China Mobile or China Unicom. It may vary due to different cards.
Password For the general cards, the password of China Telecom is vnet.mobi or card. There is no password for China Mobile or China Unicom.It may vary due to different cards.
Update Period 0~7200s. Default setup is 30s. Alive time means for the non-dial period, system can get 4G dial status each 30s to disconnect the dial operation to save the flux.
Time Range It is to set dial period. You can use message activation if there is no period setup to send out message to the SIM card of the device to enable/disable dial function.
Wireless Network Status
IP Address It is to display IP after 4G dial succeeded.
Wireless Signal
Signal It is to display intensity of current signal. Dial operation may fail, or system can not send out data even dial operation succeeded, if the signal intensity is too low.

Mobile Setup

Message Send Please go to the Event Manager interface to check the corresponding message activation item(s), then system can send out message to the specified mobile number when an event occurred. If you want to use this function please check the message activation function on the Event Manager interface.
Message Activation The mobile number on the list can send out ON/OFF, REBOOT message to the SIM card of the device to enable/disable dial function or reboot device.
Phone Activation The mobile number on the list can send out message to the SIM card of the device to enable dial function.

Note：
1. SIM card does not support hotswap function. Hotswap operation may burn down the SIM. You need to reboot 4G module after you insert a SIM card, otherwise system can not detect the new card.
2. System can not receive/send out message, receive incoming card or 4G dial if there is no SIM card.
