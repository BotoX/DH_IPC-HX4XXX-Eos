SMTP（Email） back list

Function

 

SMTP port: default value is 25.

Anonymity： For the server supports the anonymity function. You can auto login anonymously.

Health mail enable： This function allows the system to automatically send out the test information to check the email connection is proper or not. You need to check the box here to enable this function and set the update period.

If the update period is 0, it means no interval. System can send out an email when an alarm, video detect, abnormal event occurred.

System max supports three receiver addresses.

The test email box can receive the health email if your setup is right. Please save the email configuration information before you test the email.
