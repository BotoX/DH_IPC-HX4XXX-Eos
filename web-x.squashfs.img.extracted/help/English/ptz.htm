PTZ back list

Comment： Only the equipments which support these scales have the function.

  • 1. Fast positioning button. Use your mouse to select a zone in the monitor window, PTZ can fast rotate and focus to current position.

  • 2. Step is for speed operation. The larger the value is, the faster the speed is. The step is for PTZ direction control, zoom, focus and aperture.

  • 3. PTZ has the scan, preset, tour, pattern, AUX, wiper and light function.

  • 4. Virtual joystick is to drag the button at the centre to simulate the joystick operation to control the device movement.

  • Parameter Function
    Scan

    ■ Click "Set" button, you can set scan left and right limit.

    ■ Move the camera to you desired location and then click "Set Lift/Right" button to set the left/right limit.

    Preset

    ■ Input the preset value and then click "Goto" button, the camera turns to the corresponding position of the preset.

    ■ Use direction keys to move the camera to your desired location and then input preset value. Click "Add" button, you have set one preset.The preset value due to different protocols.

    Tour

    ■ Input tour value and then click the "Add" button, you have set one tour. The tour value due to different protocols.

    ■ Then input preset value in the column. Click "Add Preset" button, you have added one preset in the tour. Or you can click "Del Preset" button to remove one preset from the tour.

    Note: Repeat the above procedures you can add or delete more presets in one tour.

    Pattern You can input pattern value and click "Add" button, then click "Start Rec" button to begin PTZ movement such as zoom, focus, aperture, direction and etc. Please go back to this interface to click "Stop Rec" button. Now you have set one pattern.
    Assistant You can input assistant function value and then click "AUX On" or "AUX Off" button to enable or disable the corresponding assistant function.
    Light wiper You can enable or disable the light/wiper.
    Absolute Move Input the horizontal degree, vertical degree and the zoom rate. Click the "Goto" button, you can go to the accurate position. The unit of the horizontal degree, and vertical degree are 0.1°.
    Pan Start or stop the camera rotation movement.

  • Note： Please make you have set PTZ protocol before you use the PTZ control function.
