Abnormity Introduce

It includes five statuses: No SD card, capacity warning, SD card error, disconnection, IP conflict and unauthorized access.

SD Card

SD Card alarm occurred when there is no SD card, insufficient SD card space or SD card error.

Parameter Function
Enable Please check the box here to enable SD card error alarm function.
Relay-out The alarm output channel is enabled when an alarm occurs. It can activate the alarm output device. You need to check the box to enable this function.
Relay-out Delay The alarm output can delay for the specified time after alarm stops. The value ranges from 10s to 300s.

Note: For no SD card alarm, insufficient space alarm or the SD card error alarm, the alarm delaying time begins with the alarm occurrence time.

Send Email If you enable this function, system can send out email to alert the specified user. You can go to" Network->SMTP(Email) " to set the email address.
Capacity limit Here you can set SD card free percentage. System can generate an alarm when the free space is lower than the threshold you set here.

Network

System can generate a network alarm when network disconnection or IP conflict occurred.

Illegal Access

Once the login failure reaches the specified times, it may result in illegal access alarm event. It has the similar operation as the SD card.

Login Error: System generates illegal access alarm event and lock the account once the login error times reaches the threshold here.
